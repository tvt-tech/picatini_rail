# Picatinny Rail Calculator

A reticle/drop calculator for accounting for Picatinny rail angle, built with React, Vite, and Tailwind CSS.

## Get started

1. Install dependencies

   ```bash
   yarn install
   ```

2. Start the dev server

   ```bash
   yarn dev
   ```

3. Build for production

   ```bash
   yarn build
   ```

   The output is written to `dist/`.
